Chapter X & XI

HSC & Maritime Security

CH X:
High-Speed Craft (HSC)

ข้อกำหนดสำหรับเรือความเร็วสูง:

  • ต้องปฏิบัติตาม HSC Code ซึ่งเป็นกฎเกณฑ์ที่เข้มงวดกว่าเรือทั่วไป ทั้งด้านโครงสร้าง อุปกรณ์ช่วยชีวิต และความปลอดภัยของระบบนำร่อง.
  • เน้นเรื่องความปลอดภัยของผู้โดยสารเนื่องจากความเร็วที่สูงมาก การชนเพียงเล็กน้อยอาจส่งผลกระทบรุนแรง.

CH XI-2:
Maritime Security (ISPS Code)

ISPS Code (ความปลอดภัยทางทะเล):

  • SSO (Ship Security Officer): ผู้รับผิดชอบความปลอดภัยบนเรือ.
  • SSP (Ship Security Plan): แผนรักษาความปลอดภัยที่ต้องได้รับการอนุมัติ.
  • Security Levels: การปรับระดับความปลอดภัย (Level 1, 2, 3) ตามสถานการณ์ภัยคุกคาม.
