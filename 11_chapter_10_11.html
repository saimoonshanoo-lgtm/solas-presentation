Chapter X & XI

HSC, Safety & Maritime Security

เรือความเร็วสูง และมาตรการพิเศษด้านความปลอดภัย/ความมั่นคง

CH X:
High-Speed Craft (Reg 1-3)

ข้อกำหนดสำหรับเรือความเร็วสูง (HSC - High-Speed Craft):

  • เรือโดยสารหรือเรือสินค้าที่มีความเร็วสูงมาก ต้องปฏิบัติตาม HSC Code (1994 หรือ 2000) แทนข้อกำหนดบางส่วนใน Chapter I-IV.
  • รหัสนี้ครอบคลุมการออกแบบโครงสร้าง อุปกรณ์ช่วยชีวิต และระบบนำร่องที่ต้องตอบสนองฉับไว เพื่อรองรับความเสี่ยงจากการเดินเรือด้วยความเร็วสูง.

CH XI-1:
Special Measures - Safety

มาตรการพิเศษเพื่อความปลอดภัยทางทะเล:

  • IMO Number (Reg 3): เรือทุกลำต้องแสดงหมายเลข IMO อย่างถาวรที่ตัวเรือและห้องเครื่อง (ตัวเลขนี้จะติดเรือไปตลอดจนกว่าจะทำลายทิ้ง).
  • CSR (Reg 5): Continuous Synopsis Record เป็นเหมือน "สมุดประวัติถาวร" ของเรือ บันทึกประวัติการเปลี่ยนเจ้าของ, การเปลี่ยนธง, หรือ Class ต้องเก็บต้นฉบับไว้บนเรือเสมอ.

CH XI-2:
Maritime Security (ISPS Code)

ความมั่นคงทางทะเล (Reg 3 & 6):

  • ISPS Code (Reg 3): บังคับใช้รหัสว่าด้วยการรักษาความปลอดภัยของเรือและท่าเรือ.
    - SSO (Ship Security Officer): นายทหารรับผิดชอบบนเรือ.
    - SSP (Ship Security Plan): แผนรักษาความปลอดภัยที่ปกปิดเป็นความลับ.
    - Security Levels (1-3): ระดับการเฝ้าระวังและป้องกันภัยคุกคาม.
  • SSAS (Reg 6): Ship Security Alert System ปุ่มกดส่งสัญญาณขอความช่วยเหลือแบบเงียบ (Silent Alarm) ไปยังรัฐเจ้าของธงเมื่อถูกโจรสลัดโจมตี.
