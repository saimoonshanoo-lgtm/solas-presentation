Chapter XV

Safety Measures for Ships Carrying Industrial Personnel

บุคลากรทางอุตสาหกรรม (IP Code - Industrial Personnel Code)

CH XV:
Overview & Application (Reg 1-3)

ทำไมต้องมีกฎเกณฑ์สำหรับกลุ่มนี้?

  • อุตสาหกรรมพลังงานนอกชายฝั่ง (เช่น กังหันลมกลางทะเล, แท่นขุดเจาะ) มีความจำเป็นต้องขนส่งช่างเทคนิคและวิศวกรไปปฏิบัติงานกลางทะเล.
  • บุคลากรเหล่านี้ ไม่ใช่ "ผู้โดยสารทั่วไป" (เพราะต้องทำงานบนโครงสร้างกลางทะเล) และ ไม่ใช่ "ลูกเรือ" (เพราะไม่ได้ทำหน้าที่เดินเรือ).
  • Application (Reg 1): บังคับใช้กับเรือสินค้าและเรือความเร็วสูง (ขนาด 500 GT ขึ้นไป) ที่ขนส่งบุคลากรทางอุตสาหกรรม "มากกว่า 12 คน".

CH XV:
IP Code Requirements (Reg 4)

หัวใจสำคัญของ IP Code (Reg 4):

  • Safety Certificate: เรือต้องได้รับการตรวจสอบและออกใบรับรอง "Industrial Personnel Safety Certificate" ควบคู่ไปกับใบรับรองเรือสินค้าปกติ.
  • Training & Medical: บุคลากร (IP) ทุกคนต้องมีใบรับรองแพทย์ และผ่านการฝึกความปลอดภัยพื้นฐาน (Basic Safety Training) คล้ายคลึงกับคนประจำเรือก่อนลงเรือ.
  • Safe Transfer: ต้องมีมาตรการความปลอดภัยสูงสุดในขั้นตอนการถ่ายเทคน (Personnel Transfer) จากเรือไปยังโครงสร้างนอกชายฝั่ง.
